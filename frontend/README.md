## Getting Started

1. Run `npm install`
2. Run `npm run dev`

## Google login (optional)

This frontend can use Google Identity Services for a "Continue with Google" login.

1. Create `frontend/.env.local`
2. Add:

	`VITE_GOOGLE_CLIENT_ID=YOUR_GOOGLE_OAUTH_CLIENT_ID`
